كيفية الدخول على الراوتر لجميع الأجهزة (TP-Link / Huawei / WE / Vodafone / STC)

أسرع طريقة: افتح المتصفّح واكتب عنوان الراوتر (IP الافتراضي) في شريط العنوان، ثم أدخل بيانات الدخول. إن لم تنجح، اعرف بوابة الشبكة من جهازك (Gateway) واستخدمها مباشرة.

عناوين الدخول الشائعة

الشركةعناوين دخول شائعةملاحظات
TP-Link 192.168.0.1 — 192.168.1.1
tplinkwifi.net 		موديلات جديدة تطلب إنشاء كلمة مرور في أول دخول
Huawei (4G/5G/Fiber) 192.168.8.1 — 192.168.100.1 — (5G: 192.168.3.1) كلمة المرور غالبًا على الملصق الخلفي
WE (TE-Data) 192.168.1.1 بيانات الـ Admin مطبوعة على الراوتر
Vodafone 192.168.1.1 — 192.168.0.1 قد تظهر صفحة بنطاق خاص مثل vodafone.station
STC Fiber 192.168.1.1 — 192.168.100.1 موديلات GPON (هواوي/نوكيا…)

خطوات عامة (لكل الراوترات)

  • اتصل بالراوتر عبر كابل LAN أو Wi-Fi الخاص به (وليس نقطة اتصال الهاتف).
  • افتح المتصفح واكتب عنوان الدخول (IP) بالأعلى، واضغط Enter.
  • أدخل اسم المستخدم وكلمة المرور الافتراضية (موجودة على ملصق الراوتر) أو التي قمت بتغييرها سابقًا.
  • إن لم يفتح العنوان، اعرف بوابة الشبكة ثم استخدمها:
    • Windows: افتح موجه الأوامر واكتب: ipconfig ثم خذ قيمة Default Gateway.
    • macOS: في الطرفية: route -n get default | grep gateway
    • Linux: في الطرفية: ip route | grep default

الدخول على راوتر TP-Link

  • جرّب tplinkwifi.net أو 192.168.0.1 أو 192.168.1.1.
  • الموديلات الحديثة تطلب إنشاء كلمة مرور Admin عند أول دخول؛ القديمة كانت افتراضيًا admin / admin.
  • لو كان الجهاز من سلسلة Deco، فالإدارة تتم عبر تطبيق TP-Link Deco.

الدخول على راوتر WE (TE-Data)

  • افتح 192.168.1.1.
  • اسم المستخدم غالبًا admin، وكلمة المرور موجودة على ملصق الجهاز (أحيانًا الرقم التسلسلي أو كلمة مطبوعة).
  • بعد الدخول يُفضّل تغيير كلمة مرور الواي-فاي وكلمة مرور الـ Admin.

الدخول إلى راوتر Huawei

  • جرّب 192.168.8.1 (راوترات 4G/5G) أو 192.168.100.1 (ألياف).
  • البيانات الافتراضية عادة admin وكلمة المرور مطبوعة على الجهاز. في بعض الموديلات قد تكون أول دخول لإنشاء كلمة جديدة.

الدخول على راوتر Vodafone

  • جرّب 192.168.1.1 أو 192.168.0.1.
  • قد يظهر نطاق خاص مثل vodafone.station. بيانات الدخول الافتراضية على الملصق الخلفي.

الدخول على راوتر STC (ألياف)

  • جرّب 192.168.1.1 وإن لم يفتح فغالبًا 192.168.100.1 على موديلات GPON (هواوي/نوكيا…)
  • اسم المستخدم عادة admin وكلمة المرور على الملصق. بعض الموديلات قد تستخدم بيانات مختلفة مطبوعة بوضوح.

حل مشكلة عدم فتح صفحة الراوتر

  • تأكد أنك متصل بشبكة الراوتر نفسه (وليس بيانات الهاتف أو شبكة أخرى).
  • استخدم بروتوكول http (وليس https): مثل http://192.168.1.1.
  • اعرف بوابة الشبكة (Gateway) من جهازك واستخدمها كما بالخطوات العامة.
  • جرّب متصفحًا آخر، أو امسح الكاش: Windows → ipconfig /flushdns.
  • أعد تشغيل الراوتر والكمبيوتر. إن لم تتذكر كلمة المرور، يمكن عمل Reset بالزر الخلفي (يعيد الإعدادات الافتراضية).
